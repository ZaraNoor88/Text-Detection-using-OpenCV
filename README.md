# Text-Detection-using-OpenCV
This repository demonstrates text detection and recognition from images using **EasyOCR** and **OpenCV**.

## Features
- Detects English text from images
- Draws bounding boxes around detected text
- Displays confidence scores
- Saves output image with annotations

## Applications:
Text detection is widely used across different fields like **Autonomous Vehicles, Healthcare, Surveillance and Security, Retail and E-Commerce and Intelligent Transportation Systems.**
